# PlayUp — фотобанк

Lifestyle-фотографии для клиентского приложения и админки. Соответствуют
фотостилю из `25_playup_design_system_style.md`: тёплый золотой свет,
приглушённые тона, плёночность, «принадлежность, а не спорт».

Все файлы — PNG, 1122×1402 (вертикаль 4:5), подходят под мобильные hero-блоки
и карточки клиента.

## Использование

- Источник правды по обработке/тону — `25_playup_design_system_style.md`.
- При вставке в client/admin приложения брать копии отсюда; этот каталог —
  исходники бренда. Перед использованием в проде ужать (WebP/AVIF) и нарезать
  под нужные размеры.
- Имена файлов стабильные — на них можно ссылаться.
