# 15. Admin Screens Detailed Spec

## Назначение

Этот файл описывает экраны админки на уровне, достаточном для начала разработки интерфейса.

Админка — это ежедневное рабочее место команды PlayUp. Главная задача — быстро создавать события, видеть записи, отмечать явку, фиксировать оплаты, вносить расходы и понимать состояние бизнеса.

## Общий layout

Слева — меню. Сверху — короткая панель с названием раздела, основным действием и фильтрами. В центре — рабочая область.

Меню MVP:

- Главная
- Пользователи
- Расписание
- Площадки
- Хосты
- Финансы
- Пуши и опросы
- Аналитика
- Настройки

Если нужно упростить, “Хосты” можно спрятать в “Настройки”, а “Аналитику” частично встроить в Главную и Финансы.

## 1. Главная

Цель: показать ближайшие события и срочные действия.

Блок “Ближайшие события”:

- дата;
- время;
- название;
- активность;
- площадка;
- хост;
- записано / вместимость;
- кворум;
- ожидаемая выручка;
- статус риска;
- действие.

Статусы риска: меньше кворума, ок, почти заполнено, заполнено, прошло и не закрыто.

Блок “Действия”:

- отметить явку;
- отметить оплаты;
- отправить напоминание;
- отправить опрос;
- внести расходы;
- событие ниже кворума;
- есть неоплаченные участники.

Блок “Метрики недели”:

- выручка;
- расходы;
- прибыль;
- новые пользователи;
- пришли впервые;
- пришли повторно;
- no-show rate.

Empty state: если событий нет, показать кнопку “Создать событие”.

## 2. Пользователи

Список пользователей — таблица:

- имя;
- Telegram;
- статус;
- интересы;
- уровень;
- источник;
- посещения;
- no-show;
- последняя активность;
- дата регистрации.

Фильтры: статус, активность, уровень, источник, есть no-show, период регистрации.

Поиск: имя, Telegram username, телефон.

Действия: открыть карточку, отправить сообщение, записать на событие.

## 3. Карточка пользователя

Вкладки: Обзор, События, Оплаты, Опросы, Заметки.

Обзор:

- имя;
- Telegram ID;
- Telegram username;
- телефон;
- уровень;
- интересы;
- район;
- источник;
- статус;
- дата регистрации;
- последняя активность.

Метрики:

- посещений всего;
- записей всего;
- no-show;
- текущая серия недель;
- суммарно оплачено;
- любимая активность.

Действия: редактировать, записать на событие, отправить сообщение, добавить заметку.

## 4. Расписание

Вид по умолчанию — неделя.

Колонки — дни недели. Строки — часы. Карточка события показывает время, название, активность, записано / вместимость, статус.

Действия:

- клик по пустому слоту — создать событие;
- клик по событию — открыть карточку;
- кнопка “Создать событие”;
- переключатель “Календарь / Список”.

Список событий:

- дата;
- время;
- название;
- активность;
- площадка;
- хост;
- статус;
- записи;
- явка;
- выручка.

Фильтры: дата, статус, активность, площадка, хост.

## 5. Карточка события

Верхняя часть:

- название;
- статус;
- дата и время;
- активность;
- площадка;
- хост;
- цена;
- вместимость;
- кворум;
- записано;
- пришло;
- оплачено;
- выручка;
- расходы;
- прибыль.

Кнопки: редактировать, опубликовать, отменить, завершить, отправить сообщение, отправить опрос.

Вкладка “Участники”:

- имя;
- Telegram;
- статус записи;
- явка;
- оплата;
- сумма;
- касса;
- комментарий.

Быстрые действия: добавить участника, отметить всех пришедшими, отметить выбранных no-show, отметить выбранных оплатившими, применить цену события, экспорт CSV.

Вкладка “Финансы”: доходы, расходы, прибыль, список транзакций, добавить расход, добавить доход.

Вкладка “Опросы”: средняя оценка, ответы, текстовые комментарии.

## 6. Площадки

Список:

- название;
- адрес;
- контакт;
- стандартная цена;
- статус;
- событий всего;
- расходов всего.

Карточка: базовая информация, события, транзакции, заметки.

## 7. Хосты

Список:

- имя;
- Telegram;
- телефон;
- стандартная оплата;
- событий проведено;
- средняя оценка;
- статус.

Карточка: контакты, оплата, события, расходы, оценки, заметки.

## 8. Финансы

Главный экран:

- доход за период;
- расход за период;
- прибыль;
- баланс по кассам;
- неоплаченная сумма.

Таблица транзакций:

- дата;
- тип;
- сумма;
- касса;
- категория;
- событие;
- площадка;
- хост;
- пользователь;
- описание.

Действия: добавить доход, добавить расход, добавить кассу, экспорт CSV.

## 9. Пуши и опросы

Ручная отправка:

- сегмент;
- событие, если нужно;
- текст;
- предпросмотр получателей;
- отправить.

Автосообщения MVP: за 24 часа, за 3 часа, после события.

Опросы: список, создание, ответы.

## 10. Аналитика

Блоки: события, пользователи, деньги, повторные визиты, источники.

Фильтры: период, активность, площадка, хост, источник.

## 11. Настройки

Разделы: активности, уровни, источники привлечения, категории финансов, кассы, админы.

## Приоритет экранов

Сначала: Главная, Расписание, Карточка события, Пользователи, Карточка пользователя, Финансы.

Потом: Пуши, Опросы, Площадки, Хосты, Аналитика, Настройки.
